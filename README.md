✈️ TravelGo – Travel Booking Web Application

TravelGo is a Flask-based web application that allows users to explore travel destinations and book trips online. The system provides features such as user authentication, seat selection, travel scheduling, and booking history management.

🌟 Features :

    - User registration and login

    - Travel booking system

    - Seat selection and travel timing

    - Ticket pricing in Indian currency (₹)

    - Booking history tracking

    - Payment page UI

    - Popular destinations section for travel inspiration

🛠️ Technologies Used :

    - Python

    - Flask

    - Flask-SQLAlchemy

    - SQLite

    - HTML, CSS, JavaScript

    - Cloud Integration (Planned)

    - Amazon EC2 – Application hosting

    - Amazon DynamoDB – Cloud database

    - Amazon Simple Notification Service – Booking notifications

    - Amazon Web Services – Cloud platform

⚙️ Installation :

Clone the repository

git clone https://github.com/YOUR_USERNAME/travelgo-booking-system.git

Navigate to the project directory

cd travelgo-booking-system

Install dependencies

pip install -r requirements.txt

Run the application

python app.py

Open in browser

http://10.28.178.69:5000

🗂️ Project Structure

    - app.py               # Main Flask application

    - models.py            # Database models

    - templates/           # HTML templates

    - static/              # CSS, images, JavaScript

    - requirements.txt     # Project dependencies
